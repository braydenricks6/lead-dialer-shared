Assign a state to any number below. When this is on, calling a lead in that state automatically goes out from that number — a Florida lead sees your Florida number. Spam Registration

Carriers flag brand-new numbers as spam by default until they're registered as a real business. Register every number the same day you buy it, before you start dialing hard on it — registering after it's already flagged doesn't reliably undo the label.
Both links are free. Free Caller Registry registers you with all three carrier networks at once; Hiya's dashboard lets you check the resulting reputation grade for a number after a few days of calling.

Auto-Dial Defaults

Default queue:
One rest per attempt, comma-separated: "1, 3, 7, 14" = after the 1st try the lead rests a day, after the 2nd it rests 3 days, then 7, then 14 for every try beyond (last number repeats). Resting leads leave the queue and resurface on their own — no manual re-filtering. Due callbacks always jump the fence, and "Stop after" archives never-reached leads once they've had that many dials.
Dials one lead at a time with answering-machine detection. A machine pickup is auto-marked Straight to Voicemail, a full ring with no pickup is auto-marked No Answer, and it keeps going on its own — it only rings your browser (with a beep) when a real person answers. Walk away and come back when you hear the beep. If retrying more than once, a machine pickup on the first try still counts toward the total unless you check the box above — a voicemail box won't become a person on the next attempt.

Federal telemarketing rules restrict calls to 8am–9pm in the lead's time zone. This filters the auto-dial queue only — the manual Call button always works, for callbacks and leads you already have permission to reach outside this window.

Dispositions

"Reached" marks the lead as spoken-to (green dot) — this alone already keeps them out of the queue whenever Auto-Dial is set to "Haven't reached." "Skip in auto-dial" is a second, stronger filter: it removes that disposition's leads from the queue no matter which "who" option is selected. If a disposition is already marked Reached, toggling Skip usually won't change the count below — it's already excluded.
